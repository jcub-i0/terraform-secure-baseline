# tf-secure-baseline Validation Report Template

## Purpose

This template provides a client-facing or internal handoff format for documenting validation results from one generated `tf-secure-baseline` validation evidence package.

Use this template for one of the actual validation scopes produced by the project:

```text
workload-bootstrap
workload-baseline
control-plane
```

This template is intended to summarize:

- which validation scope was reviewed
- when validation was performed
- which export script generated the evidence package
- which automated validation checks passed or failed
- which evidence files were generated
- which manual validation activities remain
- any warnings, exceptions, or limitations

This template does not imply that a combined report exporter exists. The project currently generates separate evidence packages for workload bootstrap, workload baseline, and control-plane validation.

```bash

NAME_PREFIX="${NAME_PREFIX:-${CLOUD_NAME}-${ENV_NAME}}"
```

Control-plane export derives `NAME_PREFIX` from `CLOUD_NAME` and `CONTROL_PLANE_ENV_NAME` unless `NAME_PREFIX` is explicitly overridden:

```bash
CONTROL_PLANE_ENV_NAME="${CONTROL_PLANE_ENV_NAME:-control-plane}"
CLOUD_NAME="${CLOUD_NAME:-tf-secure-baseline}"
NAME_PREFIX="${NAME_PREFIX:-${CLOUD_NAME}-${CONTROL_PLANE_ENV_NAME}}"
```

Use `not-applicable` for fields that do not apply to the selected validation scope.

## Limitations

This report validates deployed AWS control presence and selected configuration settings for the selected validation scope.

The validation scripts confirm the presence and configuration of selected AWS security controls, governance resources, state backend resources, GitHub OIDC resources, and supporting infrastructure at the time validation was run.

This report does not replace:

- A full SOC 2 audit
- A full ISO 27001 audit
- Control owner review
- Policy and procedure review
- Control design assessment
- Control operating effectiveness testing
- Formal audit evidence review
- Risk assessment
- Vendor risk review
- Incident response testing
- Business continuity or disaster recovery testing
- A complete Information Security Management System
